# Chicken Detection and Behaviour Analysis using YOLO

# Project Overview
This project focuses on detecting chickens and analyzing their behavior using computer vision techniques. The system uses the YOLO (You Only Look Once) object detection model to identify chickens in images and videos and analyze their movement and activity patterns.

The goal of the project is to assist in poultry farm monitoring by automatically detecting chickens and observing their behavior for better farm management and animal health monitoring.

# Dataset
The dataset was collected and prepared using Roboflow. Images were annotated and organized for training the object detection model.

Dataset preprocessing included:
- Image annotation
- Data augmentation
- Dataset splitting (training and validation)

# Model
- Model Used: YOLO (You Only Look Once)
- Task: Object Detection
- Training Platform: Python environment
- Training Data: Roboflow dataset
- Detection Input: Images and Video

# Features
- Chicken detection in images
- Chicken detection in video streams
- Real-time object detection using YOLO
- Behavior monitoring based on detected chicken movements

# Technologies Used
- Python
- YOLO
- OpenCV
- Roboflow
- NumPy
- Matplotlib

# Project Workflow
1. Dataset collection and annotation using Roboflow
2. Dataset preprocessing and augmentation
3. Training the YOLO model
4. Detecting chickens in images
5. Detecting chickens in video streams
6. Behavior observation and analysis

# Results
The trained model successfully detects chickens in both images and videos and can be used for automated poultry monitoring systems.
